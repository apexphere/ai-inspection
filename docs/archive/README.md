# Archived Documentation

This folder contains superseded or outdated documentation preserved for historical reference.

## Why Archive?

- **Audit trail** — Track how designs evolved
- **Context** — Understand past decisions
- **Recovery** — Restore if needed

## Naming Convention

Archived files include the date and reason:
```
{original-name}.archived-{YYYY-MM-DD}.md
```

## Restoring

To restore a doc, move it back and remove the `.archived-*` suffix.
